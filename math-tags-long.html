Limits

To put limit notation into an expression, use the command \lim followed by a subscript, as shown below.

What you type \lim_{x \rightarrow c} f(x) = L
How it will be rendered $$\lim_{x \rightarrow c} f(x) = L$$

While subscripts are normally rendered to the bottom right of the preceding symbol, when written after a \lim command, a subscript is automatically placed underneath the limit sign.

Use the command \infty to write the infinity symbol.

What you type \lim_{x \rightarrow \infty} f(x) = L
How it will be rendered $$\lim_{x \rightarrow \infty} f(x) = L$$

To denote the limit inferior and the limit superior of a sequence, use the commands \liminf and \limsup.

What you type \liminf_{n \rightarrow \infty} x_{n}
How it will be rendered $$\liminf_{n \rightarrow \infty} x_{n}$$
What you type \limsup_{n \rightarrow \infty} x_{n}
How it will be rendered $$\limsup_{n \rightarrow \infty} x_{n}$$

You can also use the commands \inf and \sup for just the abbreviations 'inf' and 'sup' on their own.

Sums and Products

The summation sign is of course a capital letter sigma, but since it's drawn larger than a normal sigma, we can't just use the command \Sigma. Instead, use the command \sum. Use a subscript and a superscript to denote the lower and upper bounds of the sum.

What you type \sum_{i = m}^{n} a_{i}
How it will be rendered $$\sum_{i \,=\, m}^{n} a_{i}$$

Similarly, the product sign is a capital letter pi, but it's drawn larger than a normal pi, so use the command \prod.

What you type \prod_{i = m}^{n} a_{i}
How it will be rendered $$\prod_{i \,=\, m}^{n} a_{i}$$

Derivatives

When trying to add a derivative to a mathematical expression, it would be tempting to type the following.

What you type \frac{dx}{dt}
How it will be rendered $$\frac{dx}{dt}$$

However, in a derivative, the d's aren't supposed to be italicised, because they aren't identifiers. In order to get the d's to be non-italicised, use the command \mathrm{d}, as below.

What you type \frac{\mathrm{d} x}{\mathrm{d} t}
How it will be rendered $$\frac{\mathrm{d} x}{\mathrm{d} t}$$

Use superscripts to denote higher-order derivatives.

What you type \frac{\mathrm{d}^{2} x}{\mathrm{d} t^{2}}
How it will be rendered $$\frac{\mathrm{d}^{2} x}{\mathrm{d} t^{2}}$$

That's how to write full derivatives. To write partial derivatives, use the command \partial instead of \mathrm{d} to get the curly d.

What you type \frac{\partial f}{\partial t}
How it will be rendered $$\frac{\partial f}{\partial t}$$

Integrals

Use the command \int to write the integral sign. As with derivatives described above, the d of the infinitesimal element needs to be non-italicised. To do this, use \mathrm{d}.

What you type \int p(x) \mathrm{d} x
How it will be rendered $$\int p(x) \, \mathrm{d} x$$

Use subscripts and superscripts to denote the limits of an integral.

What you type \int_{0}^{3} p(x) \mathrm{d} x
How it will be rendered $$\int_{0}^{3} p(x) \, \mathrm{d} x$$

You can use the commands \iint and \iiint for double and triple integrals.

What you type \iint p(x, y) \mathrm{d} x \mathrm{d} y
How it will be rendered $$\iint p(x, y) \, \mathrm{d} x \, \mathrm{d} y$$
What you type \iiint p(x, y, z) \mathrm{d} x \mathrm{d} y \mathrm{d} z
How it will be rendered $$\iiint p(x, y, z) \, \mathrm{d} x \, \mathrm{d} y \, \mathrm{d} z$$

You can use the commands \oint and \oiint for single and double closed integrals.

What you type \oint p(x) \mathrm{d} x
How it will be rendered $$\oint p(x) \, \mathrm{d} x$$
What you type \oiint p(x, y) \mathrm{d} x \mathrm{d} y
How it will be rendered $$\oiint p(x, y) \, \mathrm{d} x \, \mathrm{d} y$$

Vectors

In digital and print media, vectors are, by convention, styled using bold font. To put something in bold font, use the command \textbf.

What you type \textbf{v}
How it will be rendered $$\textbf{v}$$

Put any subscripts and superscripts outside of the brackets of the \textbf command.

What you type \textbf{v}_{x}, \textbf{v}_{y}
How it will be rendered $$\textbf{v}_{x}, \textbf{v}_{y}$$

Unit vectors are denoted by a circumflex, or 'hat', over the identifier. Use the \hat command to do this. Furthermore, typically, for i and j unit vectors, the dot above each letter is removed. Use the commands \i and \j to get these variants of the letters.

What you type \hat{\textbf{\i}}
How it will be rendered $$\hat{\textbf{\i}}$$
What you type \hat{\textbf{\j}}
How it will be rendered $$\hat{\textbf{\j}}$$
What you type \hat{\textbf{k}}
How it will be rendered $$\hat{\textbf{k}}$$

Use the \cdot command to denote the dot product.

What you type \textbf{A} \cdot \textbf{B}
How it will be rendered $$\textbf{A} \cdot \textbf{B}$$

Use the \times command to denote the cross product.

What you type \textbf{A} \times \textbf{B}
How it will be rendered $$\textbf{A} \times \textbf{B}$$

You can add the nabla symbol with the command \nabla.

What you type \nabla \cdot \textbf{A}
How it will be rendered $$\nabla \cdot \textbf{A}$$
