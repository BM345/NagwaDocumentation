Derivatives

When trying to add a derivative to a mathematical expression, it would be tempting to type the following.

What you type \frac{dx}{dt}
How it will be rendered $$\frac{dx}{dt}$$

However, in a derivative, the d's aren't supposed to be italicised, because they aren't identifiers. In order to get the d's to be non-italicised, use the command \mathrm{d}, as below.

What you type \frac{\mathrm{d} x}{\mathrm{d} t}
How it will be rendered $$\frac{\mathrm{d} x}{\mathrm{d} t}$$

Use superscripts to denote higher-order derivatives.

What you type \frac{\mathrm{d}^{2} x}{\mathrm{d} t^{2}}
How it will be rendered $$\frac{\mathrm{d}^{2} x}{\mathrm{d} t^{2}}$$

That's how to write full derivatives. To write partial derivatives, use the command \partial instead of \mathrm{d} to get the curly d.

What you type \frac{\partial f}{\partial t}
How it will be rendered $$\frac{\partial f}{\partial t}$$

Integrals

Use the command \int to write the integral sign. As with derivatives described above, the d of the infinitesimal element needs to be non-italicised. To do this, use \mathrm{d}.

What you type \int p(x) \mathrm{d} x
How it will be rendered $$\int p(x) \, \mathrm{d} x$$

Use subscripts and superscripts to denote the limits of an integral.

What you type \int_{0}^{3} p(x) \mathrm{d} x
How it will be rendered $$\int_{0}^{3} p(x) \, \mathrm{d} x$$

You can use the commands \iint and \iiint for double and triple integrals.

What you type \iint p(x, y) \mathrm{d} x \mathrm{d} y
How it will be rendered $$\iint p(x, y) \, \mathrm{d} x \, \mathrm{d} y$$
What you type \iiint p(x, y, z) \mathrm{d} x \mathrm{d} y \mathrm{d} z
How it will be rendered $$\iiint p(x, y, z) \, \mathrm{d} x \, \mathrm{d} y \, \mathrm{d} z$$
