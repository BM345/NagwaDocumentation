Adding chemical equations to questions

This document describes how you can add correctly-formatted chemical equations to questions that you write.

The <chem> Tag

In order to identify the chemical equations within the text of a question, enclose them in <chem> tags.

Place an 'opening' tag just before each equation, and a 'closing' tag just after each equation, as shown in the example below.

<chem>CO2 + C -> 2 CO</chem>

The opening tag is just the word "chem" with angle brackets on either side: <chem>. The closing tag is the same but with a forward slash before the word "chem": </chem>.

The chemical equation enclosed by the tags needs to be written in a special syntax, which is described in the rest of this document.

Chemical Formulae

When writing chemical formulae within a <chem> tag, element symbols can just be written as-is. The example below shows the chemical formula for carbon monoxide.

What you type CO
How it will be rendered $$\ce{CO}$$

To denote the number of atoms of an element within a compound, simply write the number after the element symbol. The examples below show the chemical formulae for molecular hydrogen and sulfuric acid.

What you type H2
How it will be rendered $$\ce{H2}$$
What you type H2SO4
How it will be rendered $$\ce{H2SO4}$$

Charges

To denote a charge on an atom or molecule with a magnitude of one, write a plus or minus sign after the relevant part of the chemical formula. The charge will be rendered as a superscript, as shown in the examples below.

What you type H+
How it will be rendered $$\ce{H+}$$
What you type Cl-
How it will be rendered $$\ce{Cl-}$$

To denote a charge with a magnitude greater than one, however, use a caret (^) before the size of the charge to indicate that the number is part of the superscript and does not denote a number of atoms, as shown in the example below.

What you type Y^99+
How it will be rendered $$\ce{Y^99+}$$

Stoichiometric Numbers

Stoichiometric numbers can be written as-is before chemical formulae, as shown in the example below.

What you type 2 H2O
How it will be rendered $$\ce{2 H2O}$$

To write a variable, such as n, enclose the letter in dollar signs, as shown below.

What you type $n$ H2O
How it will be rendered $$\ce{$n$ H2O}$$

Atomic Numbers and Mass Numbers

Atomic numbers and mass numbers appear as subscripts and superscripts before an element symbol. The syntax for a subscript is an underscore (_) followed by the subscripted text in curly brackets {}. The syntax for a superscript is a caret (^) followed by the superscripted text in curly brackets {}. Use the subscript and superscript syntax just before an element symbol to denote its atomic and mass numbers, as shown below.

What you type _{90}^{227}Th
How it will be rendered $$\ce{ _{90}^{227}Th }$$

Oxidation States

To denote an oxidation state using roman numerals, use superscript notation, as shown below.

What you type Fe^{II}Fe^{III}2O4
How it will be rendered $$\ce{ Fe^{II}Fe^{III}2O4 }$$

Reactions

The set of arrows that can be used for denoting reactions is given in the table below.

A -> B $$\ce{ A -> B }$$
A <- B $$\ce{ A <- B }$$
A <-> B $$\ce{ A <-> B }$$
A <--> B $$\ce{ A <--> B }$$
A <=> B $$\ce{ A <=> B }$$
A <=>> B $$\ce{ A <=>> B }$$
A <<=> B $$\ce{ A <<=> B }$$

To put text above an arrow, write it in square brackets [] immediately after the arrow, as below.

What you type A ->[a] B
How it will be rendered $$\ce{A ->[a] B}$$

This syntax can be extended to also put something below the arrow. Write text that should go below the arrow in a second set of square brackets.

What you type A ->[a][b] B
How it will be rendered $$\ce{A ->[a][b] B}$$

To only place text below the arrow, leave the first set of brackets empty.

What you type A ->[][b] B
How it will be rendered $$\ce{A ->[][b] B}$$

Bonds

You can denote a single bond between atoms or molecules using a hyphen -, as shown below.

What you type H-H
How it will be rendered $$\ce{H-H}$$

You can denote a double bond using an equals sign =, as shown below.

What you type O=O
How it will be rendered $$\ce{O=O}$$

You can denote a triple bond using a plus sign +, as shown below.

What you type N+N
How it will be rendered $$\ce{N#N}$$
