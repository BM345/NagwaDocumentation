Adding mathematical expressions to questions

The <math> Tag

In order to identify the mathematical expressions within the text of a question, enclose them in <math> tags.

Place an 'opening' tag just before each expression, and a 'closing' tag just after each expression, as shown in the example below.

A line is described by the function <math>y = 2x + 3</math>. What is the gradient of the line?

The opening tag is just the word "math" with angle brackets on either side: <math>. The closing tag is the same but with a forward slash before the word "math": </math>.

The mathematical expression enclosed by the tags needs to be written using LaTeX syntax. For simpler expressions, like the one shown above, often no special LaTeX commands are needed, but for more complex expressions, you may need to use special LaTeX commands.

Basic Identifiers and Numbers

An identifier is a letter that's used to represent something in a mathematical expression (such as a constant or a variable). Identifiers that are Latin letters, whether lowercase (a-z) or uppercase (A-Z), can just be written as-is in the expression. Identifiers that are Latin letters are, by convention, rendered as italic text, so, for example, with the following expression, y, m, x, and c are all identifiers, so will all be rendered as italic text, as below.

What you type y = mx + c
How it will be rendered $$y = mx + c$$

Integer and decimal numbers can be written using the digits 0-9. Use a period / full-stop to write a decimal point.

What you type 234 + 56.78
How it will be rendered$$234 + 56.78$$

Basic Operators

Some basic operators, such as the plus and minus signs, don't require any special commands to write them, as below.

What you type 2 + 3 - 4 = 1
How it will be rendered$$2 + 3 - 4 = 1$$

The multiplication sign (or the times sign or the cross product sign) can be added using the \times command, as below.

What you type 2 \times 3 = 6
How it will be rendered$$2 \times 3 = 6$$

Note that using an asterisk (*) won't work. Also avoid using a lowercase x to denote multiplication, as this will be rendered as a variable.

If you want to denote multiplication using the dot operator, use the \cdot command ('cdot' is short for 'center dot').

What you type 2 \cdot 3 = 6
How it will be rendered$$2 \cdot 3 = 6$$

The division sign (also called an obelus) can be added using the \div command.

What you type 12 \div 4 = 3
How it will be rendered$$12 \div 4 = 3$$

Equalities and Inequalities

As we've already seen, the equals sign does not require a special command. The less than and greater than signs also do not require special commands, as below.

What you type x + 3 < 10
How it will be rendered$$x + 3 < 10$$
What you type y - 4 > 8
How it will be rendered$$y - 4 > 8$$

The less than or equal to and greater than or equal to signs, however, do require special commands. To write the less than or equal to sign, use the command \leq, and to write the greater than or equal to sign, use the command \geq, as below.

What you type x + 3 \leq 10
How it will be rendered$$x + 3 \leq 10$$
What you type y - 4 \geq 8
How it will be rendered$$y - 4 \geq 8$$

Fractions

You can add fractions to mathematical expressions using the \frac command. The syntax for this command is as follows.

What you type \frac{2}{3}
How it will be rendered$$\frac{2}{3}$$

The numerator of the fraction goes in the first set of brackets, and the denominator goes in the second set.

You can put any expression in the numerator and denominator, including other fractions.

What you type \frac{a + b + \frac{2}{5}}{mx}
How it will be rendered$$\frac{a + b + \frac{2}{5}}{mx}$$

Superscripts

Superscripts are used to denote exponents. You can write a superscript using the caret (^) character followed by the number or expression you want to superscript in brackets, as below.

What you type x^{2}
How it will be rendered$$x^{2}$$
What you type e^{x}
How it will be rendered$$e^{x}$$

Subscripts

Subscripts work in the same way as superscripts, but instead of the caret (^) use an underscore (_), as below.

What you type v_{x}
How it will be rendered$$v_{x}$$

You can combine subscripts and superscripts together by writing them one after another, as below.

What you type v_{x}^{2}
How it will be rendered$$v_{x}^{2}$$

Square-roots and Cube-roots

You can add square-roots to mathematical expressions using the \sqrt command. The syntax for this command is as follows.

What you type \sqrt{x}
How it will be rendered$$\sqrt{x}$$

You can add higher-degree roots by writing the degree of the root in square brackets just after the \sqrt command, as below.

What you type \sqrt[3]{x}
How it will be rendered$$\sqrt[3]{x}$$
What you type \sqrt[4]{x}
How it will be rendered$$\sqrt[4]{x}$$

Brackets

Most of the time, writing round brackets () or square brackets [] in a mathematical expression doesn't require any special commands.

What you type (x + 3) (x - 7)
How it will be rendered$$(x + 3) (x - 7)$$
What you type [a]
How it will be rendered$$[a]$$

As we've already seen, curly brackets {} are used in the LaTeX syntax to group together parts of the mathematical expression, and so aren't shown in the render. In order to get curly brackets to actually appear in the render, use the commands \{ and \}, as below.

What you type \{ a, b \}
How it will be rendered$$\{ a, b \}$$

By default, brackets won't stretch when their contents are taller than a single line, as shown in the example below.

What you type ( \frac{2}{3} )
How it will be rendered$$( \frac{2}{3} )$$

This doesn't look very good. In order to have the brackets stretch to fit their contents, use the command \left just before the opening bracket, and the command \right just before the closing bracket.

What you type \left( \frac{2}{3} \right)
How it will be rendered$$\left( \frac{2}{3} \right)$$
What you type \left[ \frac{2a}{b} \right]
How it will be rendered$$\left[ \frac{2a}{b} \right]$$
What you type \left\{ \frac{2a}{b} \right\}
How it will be rendered$$\left\{ \frac{2a}{b} \right\}$$

Greek Letters

There is a set of special commands for adding Greek letters to mathematical expressions. Most of these commands are just a back slash (\) followed by the name of the letter.

What you type \omega
How it will be rendered$$\omega$$

To write the uppercase variant of the Greek letter, use an uppercase letter at the start of the command, as below.

What you type \Omega
How it will be rendered$$\Omega$$

Note that there are some Greek letters that look the same as some Latin letters. An example of this is uppercase eta, which looks the same as uppercase H. In these cases, there is no special command for the letter - just use the Latin letter.

Trigonometric Functions

There is a set of special commands for adding correctly-formatted trigonometric functions to mathematical expressions. Function names are by convention not italicised, so you can't just write sin or cos in the expression, as the letters will be recognised as identifiers, and rendered in italics. Instead, you have to use commands like \sin and \cos, as below.

What you type \sin (\theta)
How it will be rendered$$\sin (\theta)$$
What you type \cos \theta
How it will be rendered$$\cos \theta$$

The set of commands for trigonometric functions that are available is given in the table below.

Sine \sin Arcsine \arcsin Hyperbolic Sine \sinh
Cosine \cos Arccosine \arccos Hyperbolic Cosine \cosh
Tangent \tan Arctangent \arctan Hyperbolic Tangent \tanh
Secant \sec
Cosecant \csc
Cotangent \cot Hyperbolic Cotangent \coth

Logarithms

To add a logarithm to a mathematical expression, use the syntax below.

What you type \log_{10} (x)
How it will be rendered$$\log_{10} (x)$$

The command \log produces the word 'log' in non-italic script. Use the subscript syntax (described above) to write the base of the logarithm (or leave it out if you want the base to be implicit).

You can also use the command \lg if you prefer it to \log.

What you type \lg_{10} (x)
How it will be rendered$$\lg_{10} (x)$$

To write the natural logarithm - a logarithm with a base of e, Euler's Number - use the command \ln.

What you type \ln (x)
How it will be rendered$$\ln (x)$$
