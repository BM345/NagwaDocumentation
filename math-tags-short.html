Adding mathematical expressions to questions

You can add mathematical expressions to questions that you write using <math> tags.

Place opening and closing <math> tags around each mathematical expression, like shown in the example below.

<math>y = 2x + 3</math>

The opening tag is just the word "math" with angle brackets on either side: <math>. The closing tag is the same but with a forward slash before the word "math": </math>.

The mathematical expression needs to be written using LaTeX syntax. For simpler expressions, like the one shown above, often no special LaTeX commands are needed, but for more complex expressions, you may need to use special LaTeX commands.

Basic Identifiers and Numbers

An identifier is a letter that's used to represent something in a mathematical expression (such as a constant or variable). Identifiers that use Latin letters, whether lowercase (a-z) or uppercase (A-Z), can just be written as-is in the expression. Identifiers that use Latin letters are, by convention, rendered as italic text, so, for example, with the following expression, y, m, x, and c are all identifiers, so will all be rendered as italic text, as below.

y = mx + c $$y = mx + c$$

Integer and decimal numbers can be written using the digits 0-9. Use a period / full-stop to write a decimal point.

234 + 56.78 $$234 + 56.78$$

Basic Operators

Some basic operators, such as the plus and minus signs, don't require any special commands to write them, as below.

2 + 3 - 4 = 1 $$2 + 3 - 4 = 1$$

The multiplication sign (or the times sign or the cross product sign) can be added using the \times command, as below.

2 \times 3 = 6 $$2 \times 3 = 6$$

Note that using an asterisk (*) won't work. Also avoid using a lowercase x to denote multiplication, as this will be rendered as a variable.

If you want to denote multiplication using the dot operator, use the \cdot command ('cdot' is short for 'center dot').

2 \cdot 3 = 6 $$2 \cdot 3 = 6$$

The division sign (also called an obelus) can be added using the \div command.

12 \div 4 = 3 $$12 \div 4 = 3$$

Equalities and Inequalities

As we've already seen, the equals sign does not require a special command. The less than and greater than signs also do not require special commands, as below.

x + 3 < 10 $$x + 3 < 10$$
y - 4 > 8 $$y - 4 > 8$$

The less than or equal to and greater than or equal to signs, however, do require special commands. To write the less than or equal to sign, use the command \leq, and to write the greater than or equal to sign, use the command \geq, as below.

x + 3 \leq 10 $$x + 3 \leq 10$$
y - 4 \geq 8 $$y - 4 \geq 8$$

Fractions

You can add fractions to mathematical expressions using the \frac command. The syntax for this command is as follows.

\frac{2}{3} $$\frac{2}{3}$$

The numerator of the fraction goes in the first set of brackets, and the denominator goes in the second set.

You can put any expression in the numerator and denominator, including other fractions.

\frac{a + b + \frac{2}{5}}{mx} $$\frac{a + b + \frac{2}{5}}{mx}$$

Superscripts

Superscripts are used to denote exponents. You can write a superscript using the caret (^) character followed by the number or expression you want to superscript in brackets, as below.

x^{2} $$x^{2}$$
e^{x} $$e^{x}$$

Subscripts

Subscripts work in the same way as superscripts, but instead of the caret (^) use an underscore (_), as below.

v_{x} $$v_{x}$$

You can combine subscripts and superscripts together by writing them one after another, as below.

v_{x}^{2} $$v_{x}^{2}$$

Square-roots and Cube-roots

You can add square-roots to mathematical expressions using the \sqrt command. The syntax for this command is as follows.

\sqrt{x} $$\sqrt{x}$$

You can add higher-degree roots by writing the degree of the root in square brackets just after the \sqrt command, as below.

\sqrt[3]{x} $$\sqrt[3]{x}$$
\sqrt[4]{x} $$\sqrt[4]{x}$$
